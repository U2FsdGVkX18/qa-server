# qa-server

#### 介绍

测试团队构建服务

#### 软件架构

java11 + springboot2.7.0 + gradle + mybatis-plus

#### 目录说明

```javascript
qa - server
| --commons  	        //通用(组件,配置类,工具类)
| --controller 	        //web层,启动类入口
| --domain		//实体类
| --mapper		//数据持久层,数据操作,面向数据库
| --mq			//消息中间件
| --service		//业务层,所有业务代码
| --build.gradle 			//依赖配置文件
```
